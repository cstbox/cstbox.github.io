modbus - Modbus devices support extension¶

pycstbox.modbus¶

Common definitions and helpers for Modbus devices support.

class pycstbox.modbus.ModbusRegister¶

Modbus register description.

Variables:
  • addr – register address
  • size (int) – register size (in 16 bits words)
  • cfgreg (bool) – True if this register is a configuration one

This class implements the model of the service managing the sub-network built with Modbus products.
